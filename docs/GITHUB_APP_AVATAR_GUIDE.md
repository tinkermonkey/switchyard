# GitHub App Avatar Setup Guide

## Avatar Design Concept

For the `orchestrator-bot` GitHub App, here are recommended avatar concepts:

### Concept 1: Conductor's Baton + Code (RECOMMENDED)
A stylized conductor's baton crossing with a code bracket `{ }` or `< >`, suggesting orchestration of code.

**Style**: Minimalist, flat design, works at small sizes
**Colors**:
- Primary: Deep blue (#2563EB) or purple (#7C3AED)
- Accent: Bright cyan (#06B6D4) or green (#10B981)
- Background: White or very light gray

### Concept 2: Robot Conductor
A simple, friendly robot silhouette holding a conductor's baton.

**Style**: Geometric, modern
**Colors**: Same as above

### Concept 3: Abstract "O" Logo
The letter "O" (for Orchestrator) styled as:
- Circuit board pattern inside
- Or musical staff lines
- Or interlocking gears

```

Conductor's baton crossing with code brackets { }.
Flat design, geometric, professional.

256x256 pixels.
```

### Option 2: Icon Libraries (Free)

**Sources**:
- [Heroicons](https://heroicons.com/) - MIT licensed
- [Lucide](https://lucide.dev/) - ISC licensed
- [Phosphor Icons](https://phosphoricons.com/) - MIT licensed

**Recommended icons to combine**:
- Musical note + Code
- Sparkles + Terminal
- Wand + Brackets

### Option 3: Custom Design (Professional)

**Figma/Sketch Steps**:
1. Create 256x256 artboard
2. Draw simple conductor baton (diagonal line with handle)
3. Add code brackets { } behind it
4. Use 2-3 colors max
5. Export as PNG and SVG

## GitHub App Avatar Requirements

- **Size**: 200x200 minimum, 1024x1024 recommended
- **Format**: PNG or JPG
- **Aspect Ratio**: 1:1 (square)
- **File Size**: Under 1MB
- **Background**: Transparent or solid color

## How to Upload

1. Go to GitHub → Settings → Developer settings → GitHub Apps
2. Find your `orchestrator-bot` app
3. Click "Edit"
4. Under "Display information", click "Upload a logo"
5. Select your image
6. Crop if needed
7. Save
